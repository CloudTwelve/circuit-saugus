# Circuit Saugus
This is the website draft for Circuit Saugus, a hackathon hosted at PCSSII by PCSSII Robotics for teenagers in Greater Boston.

## Contents
The contents of this website include:
  - About
    - This summarizes the hackathon
  - Schedule
    - This provides the event's (tentative) schedule
  - Speakers
    - This provides a (currently empty) list of the speakers at the hackathon
  - Sponsors
    - This provides a (currently empty) list of the hackathon sponsors and an email for sponsors to reach out to 

## Special Effects
Try moving your mouse around; you'll likely experience a surprise in the form of confetti! Hovering over certain text or images may also result in effects such as color changes. Additionally, there are many fun animations hidden throughout the site to enhance user experience.

## AI Use
No AI was used to write the code in this website, aside from trace usage from Google's AI Assistant when searching up CSS terms and the like.
